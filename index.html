      # lun's blogs

      # 关于本站

      本站主要记录笔者在 Java 学习过程中的笔记与思考，内容包括 Java 基础知识，Java 多线程，JVM，Redis，MySQL等等。

      博客的内容主要是站在巨人的肩膀上，阅读了书籍、博客，并最终记录下自己从0到1认识某个知识的过程以及过程中的一些思考，大部分内容都涉及源码的阅读。对于一些说法互有冲突的情况（例如Synchronized轻量级锁会自旋吗），本站会基于不同的观点及资料，通过阅读源码、实践等方式加以验证，最终给出笔者认为的正确答案。这当然难免有局限性，因此非常希望读者能够指正我的错误。对于一些阐述十分详细的官方回答，本站会直接引用。另外，也会包含一些随笔形式的记录性博客。

      # 勘误及提问

      由于笔者水平有限，难免出现错误或者不准确的地方。如果有疑问或者发现错误，可以在 github 相应的 Issues 进行提问或勘误。
